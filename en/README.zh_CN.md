# 简介

[平行云](https://www.pingxingyun.com/) 提供实时云渲染标准产品和丰富的接入和二次开发方式。

本着开放，易用, 服务，扩展的基本原则，完善开放接口和接入SDK。

尽可能开源不同场景下接入方式和工具链帮助开发者打造更完美的实时云渲染产品。
